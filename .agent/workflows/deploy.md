---
description: Commit and push changes
---

1. Commitni všechny změny do develop branch
2. Pushni změny na remote branch
3. Přepni se lokálně zpět do master branch
4. Vytvoř GitHub pull-request z develop do master s krátkým popisem změn
5. Mergni GitHub pull-request do master
6. Kontroluj kdy doběhne github pages deploy (pres command "gh run watch") a potom mě informuj, že stránky byly nasazeny
7. Na konci vypiš URL stránek do chatu jako odkaz
